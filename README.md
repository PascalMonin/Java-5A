## Room reservation app using Angular and Spring boot


#
### How to run the app : 
    - install docker 

```properties
 docker run --name mariadb --rm -e MYSQL_ROOT_PASSWORD=toor -e MYSQL_DATABASE=defaultdb -p 3306:3306 mariadb
```  
#### Run back :
```properties
cd ./back
```
    - run initdb scripts 
    - run application.java

#### Run front :
```properties 
cd ./front
npm install
ng serve
```
